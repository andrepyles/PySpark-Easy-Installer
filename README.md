# PySpark Easy Installer for Windows

An automated, single-click `.bat` script to set up a complete Apache Spark and PySpark environment on Windows machines.

## 🚀 Features
* Downloads and extracts **Java JDK 17**.
* Downloads and extracts the latest **Apache Spark** version dynamically from PyPI.
* Downloads **Hadoop Winutils** (`winutils.exe` and `hadoop.dll`).
* Configures system Environment Variables (`JAVA_HOME` and `PATH`).
* Installs `pyspark` and `findspark` via `pip`.

## ⚙️ How to Use
1. Download the `PySparkEasyInstaller.bat` file.
2. Double-click the file to run it.
3. Allow the Windows Administrator prompt (UAC).
4. Wait for the installation to finish and close the terminal.
5. Open a new terminal to start using PySpark!

## ⚠️ Requirements
* Windows OS.
* Python and `pip` must be previously installed.
* Active internet connection.
